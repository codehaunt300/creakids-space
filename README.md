## Penting
Ekstrak seluruh ZIP terlebih dahulu. Jangan membuka `index.html` langsung dari dalam ZIP.

Untuk GitHub Pages, upload seluruh isi folder ini dengan struktur folder tetap utuh.

Baca `ASSET-GUIDE.md` untuk ukuran pixel, nama file, cara mengganti gambar, PDF, link game, dan link YouTube.
